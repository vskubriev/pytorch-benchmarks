A set of PyTorch benchmarks.


## Usage

```
nvidia-docker run --rm -ti nerox8664/pytorch-benchmarks
```

If you want run in a loop:

nvidia-docker run --rm -ti --entrypoint "/root/loop.sh" nerox8664/pytorch-benchmarks
